# ✒️ فصيح — كاتب خطاباتك الذكي

أداة ذكاء اصطناعي عربية تتعلّم أسلوبك في الكتابة وتساعدك على صياغة الخطابات الرسمية والرسائل بأسلوبك الخاص.

## 🌐 جرّب فصيح الآن

**[افتح الموقع → faseh-app.github.io/Faseh](https://faseh-app.github.io/Faseh/)**

## ✨ المميزات

- **يتعلّم أسلوبك**: ارفع نماذج من كتاباتك السابقة (Word / PDF) ليحاكي أسلوبك.
- **ملفات أسلوب متعددة**: أنشئ أكثر من ملف أسلوب (رسمي، ودّي، إداري...) وبدّل بينها.
- **قوالب جاهزة**: احفظ قوالبك المتكررة وأعد استخدامها.
- **سجل كامل**: يحفظ كل ما كتبته للرجوع إليه لاحقًا.
- **سهل الاستخدام**: سجّل دخولك بإيميلك واسمك واستخدم البرنامج فورًا.

## 🚀 طريقة الاستخدام

1. افتح [الموقع](https://faseh-app.github.io/Faseh/).
2. سجّل دخولك بإيميلك واسمك (مرة واحدة فقط).
3. أنشئ ملف أسلوب وارفع نماذج من كتاباتك.
4. اطلب من «فصيح» كتابة خطابك بأسلوبك.

## 🔒 الخصوصية

- بياناتك الشخصية (الإيميل والاسم) تُستخدم فقط لمتابعة استخدامك للبرنامج.
- جميع ملفاتك ونصوصك وأساليبك تُحفظ محليًا في متصفحك ولا تُرفع لأي خادم.
