# ✒️ فصيح — كاتب خطاباتك الذكي

أداة ذكاء اصطناعي عربية تتعلّم أسلوبك في الكتابة وتساعدك على صياغة الخطابات الرسمية والرسائل بأسلوبك الخاص.

## ✨ المميزات

- **يتعلّم أسلوبك**: ارفع نماذج من كتاباتك السابقة (Word / PDF) ليحاكي أسلوبك.
- **ملفات أسلوب متعددة**: أنشئ أكثر من ملف أسلوب (رسمي، ودّي، إداري...) وبدّل بينها.
- **قوالب جاهزة**: احفظ قوالبك المتكررة وأعد استخدامها.
- **سجل كامل**: يحفظ كل ما كتبته للرجوع إليه لاحقًا.
- **يعمل في المتصفح بالكامل**: بياناتك تُحفظ محليًا في جهازك ولا تُرفع لأي خادم.

## 🚀 طريقة الاستخدام

1. افتح ملف `index.html` في المتصفح، أو زر النسخة المنشورة على الإنترنت.
2. أضف مفتاح Anthropic API الخاص بك (انظر قسم الإعداد بالأسفل).
3. أنشئ ملف أسلوب وارفع نماذج من كتاباتك.
4. اطلب من «فصيح» كتابة خطابك.

## ⚙️ الإعداد (مفتاح API)

يحتاج البرنامج إلى مفتاح من Anthropic لكي يعمل:

1. احصل على مفتاحك من <https://console.anthropic.com>
2. افتح ملف `index.html` وابحث عن السطر:

```js
const FASIH_API_KEY = "YOUR_ANTHROPIC_API_KEY_HERE";
```

3. استبدل النص بمفتاحك ثم احفظ الملف.

## ⚠️ تنبيه أمني مهم جدًا

هذا الموقع يتصل بواجهة Anthropic **مباشرة من المتصفح**. هذا يعني:

- **لا تضع مفتاحك الحقيقي في مستودع عام (Public)** على GitHub. أي شخص سيستطيع رؤيته ونسخه واستهلاك رصيدك.
- إذا أردت نشر الموقع للعامة مع مفتاحك، الطريقة الآمنة أن تمرّ الطلبات عبر خادم وسيط (Proxy/Backend) يخفي المفتاح، بدلًا من وضعه داخل صفحة المتصفح.
- للتجربة الشخصية أو المشاركة مع أشخاص موثوقين فقط: يمكن لكل مستخدم وضع مفتاحه الخاص بنفسه.
